JS包含三大部分：
        1：ECMAScript
            规定JS的语法规范

        2：DOM
            Document Object Model
            给我们提供了一套完整的操作页面元素的API
            (Application programming Interface)应用程序编程接口

        3：BOM
            Browser Object Model
            window.open()   打开浏览器的一个新窗口
            Navigator
            window.navigator.userAgent    获取系统的一些信息
